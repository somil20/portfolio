# Somil Mourya - Portfolio Website

A modern, responsive portfolio website for Somil Mourya, an Android & KMP Developer.

## Features

✨ **Responsive Design**
- Works seamlessly on desktop, tablet, and mobile devices
- Mobile-first approach

🌙 **Dark Mode Toggle**
- Switch between light and dark themes
- Theme preference is saved in browser

📱 **Sections**
- **Hero Section** - Eye-catching introduction
- **About Me** - Professional summary with stats
- **Skills & Technologies** - Organized skill categories
- **Featured Projects** - Showcase of main projects
- **Resume** - Download section
- **Contact** - Multiple contact methods and contact form

🎨 **Modern Features**
- Smooth animations and transitions
- Gradient backgrounds
- Floating card animations
- Scroll animations
- Active link highlighting
- Professional color scheme

## File Structure

```
portfolio/
├── index.html       # Main HTML file
├── styles.css       # Styling and responsive design
├── script.js        # JavaScript for interactivity
├── resume.pdf       # Resume file (to be added)
└── README.md        # This file
```

## Getting Started

1. Clone the repository
2. Add your resume as `resume.pdf` in the root directory
3. Update project links in the `index.html` file
4. Customize the content as needed

## Customization

### Update Your Information
- Edit the HTML content in `index.html`
- Update social media links in the contact section
- Replace project descriptions with your actual projects

### Modify Styles
- Colors can be changed in the `:root` CSS variables in `styles.css`
- Adjust spacing, fonts, and animations as needed

### Add Your Resume
- Place your resume PDF as `resume.pdf` in the root directory
- The download button will work automatically
